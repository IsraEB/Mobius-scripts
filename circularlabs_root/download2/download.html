New users should select the 2.5 release. The 2.1 release is provided only for those upgrading from 2.1 and wish to return to 2.1.

If you perform live and have been using 1.45 release, we recommend you do not upgrade unless you have the time to test with the new version. See the release notes in the 2.0 documentation for more information about what has changed.

If you have been using release 1.42 or earlier, there were many changes between 1.42 and 1.45, and many more from 1.45 to 2.0. You are encouraged to upgrade but you may have to adjust some of your scripts and redo some of your configuration.

Always make a backup of the mobius.xml and ui.xml files before performing an upgrade
